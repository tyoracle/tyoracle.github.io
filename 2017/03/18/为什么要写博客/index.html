为什么要写博客

发表于 2017-03-18 | 分类于 生涯，规划 |   |   次阅读

1

像我们做IT的，每天接触着信息量居大无比的互联网，今天的热点，明天可能就被人遗忘了。技术也一样，从PHP到java，从dos到linux，从sybase到现在的nosql。我们每天都在接触新的技术，疲于应对。

每天还需要面对工作上各种各样的琐碎小事，IT人的上班时间绝不只是上班的那个八小时，更多的是八小时以外的工作与学习。

现代大都市，上下班的时间越来越长了，当一天的工作完成，从客户处拖着疲惫的身心回到家的时候，还有一堆的文档要写，一堆的资料要翻阅，一堆的邮件等着你去回复，还要再学点什么东西的情况下，基本上没有时间了。所以你看到这个网站上的文章更新得很慢。有时在问自己，我为什么要写这个呢？

2

学生时代的语文考试，作文往往能取得不错的分数，所以很喜欢写东西，也曾投稿到全国的中学生期刊拿过所谓的三等奖，也曾做过作家梦。但毕业后，为了生计，早已忘掉读书时的梦想。有的只是社会上大多数人有着的浮燥。

随着年纪的增长，发现自己的人生已走了近半，却忘掉了自己的初衷。除了日常交付的文档，偶尔想提笔写写的时候，总是没写几句就没再写下去，以前也曾有过自己的博客，有兴趣的可以去翻阅一下：

http://blog.itpub.net/12482/

也是写了一段时间后没有继续下去。直至看到这样一句话：

尽管你有着不同凡响的愿望，但却时时刻刻像那些终究碌碌无为的人一样思考

当今社会最重要的两项基本技能是“当众演讲”与“写作”。这两个技能谁不会呢？你看，谁不会说话，谁不会写字？但是，能真正拥有懂得运用这两个技能的人有多少呢？如果真有那么多，那估计到处都是作家，都是演讲家了。那如何习得这个技能呢？只有不断去练习，不断去写，你才能把自己的文字能力提高。

如果当初你在思考的时候，用文字把它们记录下来，那么在现在，将来回顾的时候，你不光能知道当时自己的想法，更能看到自己在过去的这段时间内，是否有过深入的思考，有过进步。

思考能力，分析能力比别人要强。想要在任何一个领域成功，思考能力、分析 能力都是不可或缺的能力—这是一个铁一样的事实。

持续的写作可以锻炼一个人的学习能力、思考能力、分析能力、也是锻炼沟通能力最直接，成本最低的一种方式。这就是输出倒逼输入的一个最佳的方法。只有你持续不断去写作，你需要大量的输入，大量的做实验去证明自己的论点，这样你才能经过自己的思考，提炼出你对这个知识点的理解。

最好的学，是把一个概念教给别人，当你能把一个概念清晰，完整地讲解出来，并且被别人所接受时，那么你就完全掌握了这个概念了。你在讲解的过程中，是不是需要做大量的输入？大量的阅读相关的材料，甚至自己亲手去做做实验，你才能把这个概念的细节研究清楚，从而才能讲清楚这个概念？把一个概念教给别人，写作就是一个很好的手段，世界上没有比文字更清晰明确去表达的了。只有不断的写作，才能提高自己的文字表达能力，才能达到把概念教给别人，而让自己在教的过程中学得更好。

其实写作本身也是一种记录，以前读书的时候，我们会做笔记，但出来工作后，在你学习的时候，你记得自己做过多少笔记？所以很多东西，很多概念，在你看的时候，眼睛录入了，但是却没有真正的在脑袋里落地，也没有在脑细胞里生成连结节点，所以虽然很多概念都看过，但是真正要用的时候，却拼了命都想不起来。所以我们在看文档的时候，一定不是只用看，而应该是用笔去记录，把自己的思考写下来，然后让这些概念植根于自己的脑袋里，从而与原有知识形成连接结点，这样你就再也忘不掉了。只有这样，你的学习效率才会比别人高，从而走在了别人的前面。

3

做什么，最难的就是坚持 ，所以写也是一样的道理，只有持续不断的写，虽然现在文字很简陋，没什么人看，但有什么关系呢？只有不断去坚持，不断去思考训练，总会看到自己这一段时间的变化。其实坚持这个词，在别人看来或许是一个煲义词，但对你来说，需要坚持的东西，证明并不是你心甘情愿去做的，你只是作为一种任务，一种形式去做，并不是发自你的内心。既然如此，你也肯定 “坚持”不了多久。我们要赋予其一个意义，就是告诉自己，写作是让自己不断成长，这样，你才会主动去思考，内化为一种动力，一有空就把近期的所思所学写下来。

也许你又想问了，“那我如何开始呢？”还是那句话，这个问题没必要问。

等什么？写就是了！

写什么？写就是了！

怎么写？写就是了！

不过，倒时有个想法需要认真思考

我写的这个东西对别人有用吗？

只有有用的东西，才会帮助别人，从而体现你的价值。所以在tyoracle.com，我希望你看到的都是对你有所帮助的，特别是在数据库这一块，我会尽力把自己学习的，经历过的知识整理出来后进行分享。我做的是，不管自己写得好不好，写了有没有人看，只要是一直写下去。送自己一句话：“但行好事，莫问前程…….”
