为什么网站叫"IT职业生涯规划“

发表于 2017-02-12 |   |   次阅读

起因

这个网站域名是从去年7月份生日的时候，给自己送的一份礼物，希望能在这个网站上记录下自己的成长点滴。起初我是使用pelican进行建站，从八月开始写，也只写了七八篇的博文，甚是懒。所以今年给自己定下一个目标，一定要勤写。

转眼2017年已经过去了40多天了，但是产出还是不高。春节也过完了，自己的腰长了一圈。而年初定下的计划，是时候要实施了。

在2016，这个网站是”ORACLE星系“，可能在google上都找不到多少痕迹，因为写的东西实在太小了，为什么要叫这个名呢？这是因为，我从事ORACLE十几年，从8I开始到现在的12C，一直靠他养家活口，所以对其是十分熟悉的，所以希望将自己这十多年的经验，日常遇到的一些疑难或有趣的问题，放在这里，一个是记录自己的足迹，一个是可以分享一下自己的一些经验给行业内的新人。

但在今年年初的时候，我听到了一个故事，让我非常有感触，决定将这个网站，不再只是单纯的定位在技术网站，而是将其变成一个可以帮助行业内的新人成长与技术随笔的网站。这是一个什么样的故事呢？

我的IT之路

在讲改变我想法的故事前，先讲讲我自己的IT之路。
自1971年开始第一台个人电脑的发生，IT技术就随着PC的发展，迅猛发展。我是从96年读高中的时候，靠自己多年的零用钱，拼出了人生第一台486个人电脑。那时的电脑还动不动上万元一台，用的是5.25寸的大软盘，内存只有4M。但这台486在当时已经非常牛了，可以运行windows 3.1，有比较炫的图形了（因为当时全是DOS的时代）。那时，我会拼装电脑，那是非常酷的一件事情，天天研究CPU超频，研究那种主板好，那种显卡好（当时声卡还没有出现呢）。就是在那个时候，PC在中国的普及程度越来越快，从当时没几个人能买电脑开始，两三年后，电脑也进入了千家万户，不再是什么神秘的东西了。

随着电脑的普及，相应的工作岗位就慢慢多了起来。但那时的我，只会装机，也就只看到在电脑城，网吧的一些工作机会。并不知道外面的世界，后来，随着自己的接触的世界大了，知道了autocad，photoshop,3DMAX这些高大上的应用，自然而然也跟着去学，但因为没有参加培训班，也没有人指导，当然当时也没有网络，只能靠着一些外国翻译的书籍，就这样自学起来。

我还记得我当时觉得自己学得很不错了，会做发光的字啊，立体感的图啊：-),自我感觉特别良好，这时我觉得应该可以去找一份相关的工作了，结果呢，我还很清晰的记得自己去找的一份关于广告公司画图的工作，面试的题目就是让我现场做一个中国电信的logo出来，那时的中国电信的logo是一个中字，位移几次形成立体感效果的图，结果我花了两个小时硬是没做出来，后来人家老板就直接说，这样的水平，你还是先回去学学再来吧…..

这次应聘经历，严重的打击了我。后来，发现自己真的不是做广告创作这一块的料。再后来，互联网开始普及了，开始流行html与flash做动画，又去折腾学flash，这时候比刚开始只有书作为学习工具强多了，开始有学习光盘了，照着学习光盘折腾了一段时间后，还是没做出一个像样的动画。就这样，我折腾了两三年的时间，青春就这样悄无声息的在流逝。我找不到自己的方向了，看不清自己应该往那里去走，感觉到特别的迷茫，很想有人指导自己，肯定地告诉我，你学这个就行了，绝对没错。可时，当时没有人这样帮我。一切都靠自己跌跌撞撞在摸索，在经历过很多工作，很多次跌倒后，才走到了今天。假如那几年，有人能帮我指指路，帮我规划一下以后的职业发展方向，那就不会浪费那几年时间，我绝对可以走得更远更远。

车日路

在2009年，也就是8年前，我记得看过智联招聘的一个分析报告，说中国的IT从业人数在08年底达到了22万之多，当时觉得这是一个很大的数字，但是到现在八年后，从事IT行业的绝对是千万级的从业规模了。因为IT行业总体准入门槛不高，并且容易找到工作，所以每年都有大量的新人在进入。在这些从业人员里，有着各色各样的IT人才，如最基础的电脑城的组装工，淘宝的客服，码字的程序员，运维人员，美工，销售等等。以前我们的祖辈说365行，现在IT里就能细分出上千种工种，如运维人员可分为，系统，数据库，中间件，网络，存储等一堆，而每一项里又可以细分，如系统又可以分为：AIX,LINUX,WINDOWS,SOLARIS,HP-UNIX….
等不一一列举。在这么多职业方向里，有很多人，真的看清了自己未来的职业发展道路了吗？清晰自己真的适合自己选的方向吗?或者你自己想向那个方向发展？还是你想向这个方向发展而又不知怎么做呢？

今年我听到的一个故事，是叫车与路，是关于职业生涯的。什么是生涯呢？生涯就是帮你搞明白，你是谁，你想去那里，应该怎么去的概念。就好像我们计划了很久的一次自驾旅行，你想驾着什么车，带着什么人，走过什么样的路，去看什么样的风景，然后到达什么样的目的地呢？

其实，上述的问题，其实是与你的职业生涯相关，就是车是你的自我，有人说自己马力十足，工作喜欢有挑战，越刺激越好，有人希望自己是奔驰，喜欢在平稳的环境里慢慢开。有人喜欢自己是自行车，这样可以独自己享受旅程。而你身边的人，就是你的职业生涯中你做决策时帮你的人，是自己的家人，还是爱人，还是想一个人。而风景，就是你想经历那些事，而走什么路就是你达成目的的路径，这样，你就可以到你认为理想的职业状态。

车：自我。我是辆什么车？
日：目标。我的三到五年的目标是什么？
路：路径。如何到达？

每一个IT从业人员，有否认真思考过上述的三个问题，在夜深人静的时候，问过自己呢？只有清晰的知道自己的目标，你才能对自己的职业有一个很好的规划，从而知道通过怎样的路径去抵达这个目标。而知道了路径还不够的，你不开动自我这辆车，你只有永远停在原地，永远不会到达目的地。

是不是应该好好想想上面的问题？

正是听到了这个故事，我才有感而发，决定将这个网站改名为’IT职业生涯规划’。因为我知道，在千万的IT从业人员里面，真的有太多的新人，每年都有大量的人涌进这个行业大学生。而这些人，或者从没有想过什么样的路适合自己走，什么样的目标才是自己想要的，他们只是为了谋求一些工作，先解决生存问题。当他们工作了一段时间后，工资没法提高的时候，对当前工作失去兴趣的时候，他们就会感到非常迷茫。不知道自已这部车，想开往那个方向，走向那个目的地;又或者，看到了一个好像不错的目的地，而自己因为马力不足，而开不过去。

这个网站会以我十多年的行业经验，原厂的从业经历，告诉你，在IT的顶层圈子，你可以选择的方向，当你选择了方向后，应该选择什么样的路径，最快的到达这个目标。而当你马力不足的时候，你可以在这里学习到很多技术的知识，激励人进步的鸡汤。这样你就可以迅速升级你的汽车引擎，提升马力，更快的奔向自己的目标。

因为我本身还是在做着一线的技术，还是会以主流的技术随笔为主，但是我会穿插着自己对职业生涯规划的理解，给大家更多的建议，也希望能有更多的人能关注到这个网站。这个网站就像一棵小树苗，在慢慢成长，先给今年定个小目标，争取今年先长到一百篇。

该奋斗了
